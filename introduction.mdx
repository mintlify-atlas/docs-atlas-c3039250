## Overview

PayphoneGo is an iOS app that helps you discover and explore payphones in your area. Built with SwiftUI and MapKit, the app provides an interactive map experience with realistic elevation, location-based discovery, and seamless navigation to nearby payphones.

## Key features

<CardGroup cols={2}>
  <Card title="Interactive map" icon="map">
    View payphones on a map with realistic elevation and custom markers showing claim status
  </Card>
  <Card title="Location discovery" icon="location-dot">
    Automatically discover payphones near your current location with real-time updates
  </Card>
  <Card title="Navigation" icon="route">
    One-tap navigation to the nearest unclaimed payphone using Apple Maps
  </Card>
  <Card title="Claim history" icon="clock">
    View detailed claim history, user information, and voicemail recordings for each payphone
  </Card>
  <Card title="Smart filtering" icon="filter">
    Filter payphones by claim status to find all or only unclaimed locations
  </Card>
  <Card title="Distance tracking" icon="ruler">
    Calculate and display distance from your location to any payphone
  </Card>
</CardGroup>

## Technology stack

<Steps>
  <Step title="SwiftUI">
    Modern declarative UI framework for building native iOS interfaces
  </Step>
  <Step title="MapKit">
    Apple's mapping framework with realistic elevation and interactive controls
  </Step>
  <Step title="Swift Concurrency">
    Async/await patterns for network requests and location updates
  </Step>
  <Step title="Dependencies">
    Swift Dependencies library for dependency injection and testability
  </Step>
</Steps>

<Note>
  PayphoneGo requires iOS 26.2 or later and uses Swift 6.0 with strict concurrency checking.
</Note>
